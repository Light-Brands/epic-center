# Financial Records

Bank reconciliations, compensation reviews, housing allowance designations, and financial reporting.

## Structure

| Subdirectory | Description | Frequency |
|-------------|-------------|-----------|
| `bank-reconciliations/` | Monthly bank statement reconciliations | Monthly |
| `compensation/` | Annual compensation reviews and comparable data | Annually |
| `housing-allowance/` | Housing allowance designations (IRC 107) | Annually |
| `budgets/` | Annual church budgets | Annually |
| `tithes/` | Association tithe payments to Living Light HQ | Monthly |

## Automation

- Bank reconciliations are generated automatically when statements arrive
- Compensation review packages are prepared in October with comparable data
- Housing allowance designations are drafted in October for November board approval
- Tithe calculations are computed and reported monthly
