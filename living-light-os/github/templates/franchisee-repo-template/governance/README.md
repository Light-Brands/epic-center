# Governance Records

Board meeting minutes, resolutions, and governance policies for your church.

## Structure

| Subdirectory | Description |
|-------------|-------------|
| `board-minutes/` | Meeting minutes (quarterly + special sessions) |
| `resolutions/` | Board resolutions and formal actions |
| `policies/` | Church policies and procedures |
| `committees/` | Committee charters and meeting records |

## Board Meeting Workflow

1. **7 days before** — Living Light OS generates and sends the meeting agenda
2. **Day of meeting** — Record the meeting (audio/video)
3. **Within 48 hours** — OS drafts minutes from the recording
4. **Within 7 days** — Board reviews and approves minutes
5. **Auto-filed** — Approved minutes are committed to this directory

## Required Meetings

- **Quarterly board meetings** (minimum 4 per year)
- **Annual compensation review** (before December 31)
- **Special sessions** as needed for major decisions
