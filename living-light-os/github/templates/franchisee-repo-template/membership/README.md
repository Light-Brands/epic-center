# Membership Records

Member rolls, ordinations, covenants, and membership administration.

## Structure

| Subdirectory | Description |
|-------------|-------------|
| `rolls/` | Current and historical membership rolls |
| `ordinations/` | Minister ordination records and credentials |
| `covenants/` | Signed membership covenants |
| `certificates/` | Membership and ordination certificates |

## Membership Management

- New members sign the Church Membership Covenant
- Membership rolls are updated automatically
- Ordination candidates complete the Living Light curriculum
- All ordination credentials are issued through Living Light HQ
