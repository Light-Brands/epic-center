# Quintana Roo Property Evaluation Framework

> *Systematic Criteria for Flagship Location Selection*

---

## Purpose

This framework provides a structured methodology for evaluating specific properties in Quintana Roo, Mexico for the Transformational Epicenter flagship facility. Each property will be scored across 10 categories totaling 100 points, with clear pass/fail gates and weighted criteria aligned with operational requirements.

---

Before detailed scoring, each property must pass these non-negotiable requirements:

| Gate | Requirement | Verification Method | Pass/Fail |
|------|-------------|---------------------|-----------|
| **G1: Hospital Access** | JCI hospital reachable within 30 minutes (non-traffic) | Google Maps + physical drive test | |
| **G2: Cardiac Capability** | Hospital has cardiac catheterization lab | Hospital verification call | |
| **G3: Land Size** | Minimum 0.5 hectares (1.24 acres) usable | Survey/deed verification | |
| **G4: Zoning** | Permits residential, hospitality, or medical use | Municipal zoning verification | |
| **G5: Security Zone** | Not in US Level 3/4 advisory micro-zone | State Dept + local assessment | |
| **G6: Legal Title** | Clear title, no encumbrances, foreign ownership permitted | Legal counsel review | |
| **G7: Utilities** | Reliable power, water, internet available | Utility company verification | |

**Scoring Rule:** If ANY gate fails, property is disqualified. Do not proceed to detailed scoring.

Properties scoring 75+ should undergo full due diligence:

### Legal Due Diligence
- [ ] Title search (10+ years)
- [ ] Fideicomiso bank selection and terms
- [ ] Lien and encumbrance check
- [ ] Boundary survey
- [ ] Zoning confirmation letter from municipality
- [ ] Environmental impact assessment requirements
- [ ] Building permit pathway confirmation

### Medical Due Diligence
- [ ] Hospital site visit and administrator meeting
- [ ] Emergency transfer agreement discussion
- [ ] Ambulance response time test (actual)
- [ ] Physician credentialing pathway confirmation

- [ ] Professional property inspection
- [ ] Soil/geological assessment

- [ ] Utility capacity confirmation
- [ ] Environmental assessment (flora/fauna)
- [ ] Flood zone verification

### Financial Due Diligence

## Part 8: Evaluation Process

### Step 1: Initial Screening (Desktop)
- Review listing information
- Check Google Maps for hospital distance
- Verify area is in acceptable zone
- Preliminary pass/fail gate assessment

### Step 2: Virtual Tour/Photos
- Assess property characteristics from images
- Evaluate natural features
- Identify privacy concerns
- Preliminary scoring

### Step 3: Site Visit
- Physical verification of all criteria
- Drive time to hospitals (test actual times)
- Meet with neighbors if possible
- Assess "feeling" and healing atmosphere
- Final scoring

### Step 4: Due Diligence (Top Candidates)
- Full legal review
- Hospital partnership discussions
- Development cost estimation
- Final recommendation
